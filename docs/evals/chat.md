# Chat Evals

Chat evals cover seeded project selection, chat page rendering, session lifecycle, and streaming-related readiness.

## Current Entry Points

- `npm run eval:chat:api`
- `npm run eval:chat:smoke`

## Expected Coverage Growth

The current chat harness skeleton validates session list/create and basic chat page rendering. It should later expand to document-scope mentions, run streaming, disconnect handling, and preflight/required-input flows.
